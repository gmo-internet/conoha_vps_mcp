## ログ記録ルール

各MCPツール呼び出しごとに、以下の形式で記録する:

```
#### Step N: {操作内容}
- **ツール**: `{ツール名}`
- **パラメータ**:
  - path: `{path}`
  - param: `{param}`（該当する場合）
  - body:
    ```json
    {リクエストボディ}
    ```
- **レスポンス**:
  ```json
  {レスポンス内容}
  ```
```

バリデーションエラーやツール実行前のエラーの場合は、レスポンスの代わりにエラー内容を記録する。
